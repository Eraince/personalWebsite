This is a series of experiments on using inexpensive tools (Processing and projector) to create entertaining experience.

All backgrounds are written using Processing and then projected onto different surfaces.

I, II, III

I found a beautiful pattern from one of package boxes. I used the computer camera to get the patter and add the different color filter to create mysterious ambience. I projected this on to my closet to take a portrait shot for my friend.
